Wish List New Member Template Package.
======================================

The following fields can be referenced in the templates.
--------------------------------------------------------
  data.storeFrontHostName (string)
  data.wishListOwnerFirstName (string)
  data.wishListOwnerLastName (string)
  data.wishListId (string)
  data.wishListDisplayName (string)
  data.newMemberFirstName (string)
  data.newMemberLastName (string)
  data.recipientEmailAddress (string)
  data.newMemberProfileImageURL (string)
  data.wishListOwnerProfileImageURL (string)
  data.recipientFullName (string)
  data.organization (object) ->
    name (string)
    logoUrl (string)
